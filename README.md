## How to run

Install packages

```bash
yarn install
```

run this command

```bash
  yarn postinstall
```

compile ios

```bash
  cd ios && pod install && cd ../
```

Start the server

```bash
  yarn start
```

run on ios

```bash
  yarn ios
```
